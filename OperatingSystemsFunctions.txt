Security- the operating system uses a password protection to protect the user data and information. Futhermore, it prevents unauthorized access to programs and user data. Sometimes, the operating system will give a month of free virus-free program that protects the user computer, but some of them didn't give out any of that.
Control over system performance – Monitors overall system health to help improve performance. It records the response time between on how long is the service requests, and system response to have a complete view of the system health. This helps improve the performance by providing important information needed to find out the problems of the user computer.
Memory Management – The operating system manages the primary memory and the main memory. Main memory is made up of a large array of bytes or words where each byte or word is assigned a certain address. And main memory is a fast storage and it can accessed directly by the CPU.
